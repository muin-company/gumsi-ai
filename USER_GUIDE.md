# 검시AI 사용 가이드

검정고시 AI 튜터와 함께 효과적으로 공부하는 방법

---

## 🎯 검시AI란?

검시AI는 24시간 언제든지 질문할 수 있는 AI 기반 검정고시 학습 플랫폼입니다.

### 핵심 기능
- 💬 **AI 튜터**: 국어, 수학, 영어, 사회, 과학, 한국사 전과목 지원
- 🎓 **맞춤형 설명**: 내 수준에 맞춰 쉽게 설명
- 📚 **문제 풀이 도움**: 모르는 문제 해결
- 🔍 **개념 정리**: 이해가 안 되는 개념 설명

## 💡 효과적인 질문 방법

### ✅ 좋은 질문 예시

**개념 설명 요청:**
```
피타고라스 정리가 뭐야?
```
```
광합성 과정을 쉽게 설명해줘
```

**문제 풀이 도움:**
```
2x + 3 = 7을 어떻게 풀어?
```
```
이 문제 풀이 방법 알려줘: [문제 내용]
```

**심화 학습:**
```
르네상스가 왜 이탈리아에서 시작됐어?
```
```
미분과 적분의 차이점이 뭐야?
```

**오답 분석:**
```
이 문제를 틀렸는데, 어디서 실수한 거야?
[문제와 내 답]
```

### ❌ 피해야 할 질문

**너무 모호한 질문:**
```
❌ 수학 알려줘
✅ 인수분해 공식 알려줘
```

**답만 요구:**
```
❌ 답만 알려줘
✅ 이 문제를 어떻게 풀어야 하는지 단계별로 설명해줘
```

**여러 질문 한 번에:**
```
❌ 피타고라스 정리랑 삼각비랑 미분 다 설명해줘
✅ 먼저 피타고라스 정리부터 설명해줘
```

## 📚 과목별 활용 팁

### 국어
- 문법 개념 질문: "주어와 서술어의 관계를 설명해줘"
- 독해 전략: "이 지문의 주제를 어떻게 찾아?"
- 어휘 설명: "관용어 '발이 넓다'가 무슨 뜻이야?"

### 수학
- 공식 유도: "이차방정식 공식이 어떻게 나온 거야?"
- 문제 풀이: "이 방정식을 단계별로 풀어줘"
- 개념 연결: "함수와 그래프의 관계를 설명해줘"

### 영어
- 문법 설명: "현재완료 시제를 언제 써?"
- 단어 암기법: "accept와 except의 차이점은?"
- 독해 연습: "이 문장 구조를 분석해줘"

### 사회
- 역사 이해: "임진왜란이 왜 일어났어?"
- 개념 정리: "민주주의의 핵심 원리는 뭐야?"
- 시사 연결: "이 역사적 사건이 현대에 어떤 영향을 줬어?"

### 과학
- 원리 설명: "뉴턴의 제3법칙을 쉽게 설명해줘"
- 실험 이해: "광합성 실험을 어떻게 설계해?"
- 개념 비교: "원자와 분자의 차이점은?"

### 한국사
- 시대 구분: "고려시대의 주요 사건들을 정리해줘"
- 인물 설명: "세종대왕의 업적을 설명해줘"
- 사건 분석: "삼국통일이 가진 의미는 뭐야?"

## 🎓 학습 전략

### 1. 체계적인 학습
1. **개념 이해** → AI 튜터에게 개념 설명 요청
2. **예제 풀이** → 예제 문제 풀이 방법 학습
3. **직접 연습** → 스스로 문제 풀어보기
4. **오답 분석** → 틀린 문제 AI 튜터와 분석

### 2. 반복 학습
- 같은 개념을 다르게 질문해보기
- "다른 예시로 설명해줘"
- "더 쉬운 말로 설명해줘"

### 3. 연결 학습
- 배운 개념들 연결하기
- "이 개념과 저 개념의 관계는?"
- "실생활에서는 어떻게 쓰여?"

### 4. 능동적 학습
- 단순히 답 듣지 말고 이해하기
- "왜 그렇게 되는 거야?"
- "다른 방법은 없어?"

## ⚡ 자주 묻는 질문 (FAQ)

### Q1. AI 튜터는 24시간 이용 가능한가요?
**A:** 네! 언제든지 질문하실 수 있습니다. 새벽이든 주말이든 상관없습니다.

### Q2. 한 번에 여러 질문을 해도 되나요?
**A:** 가능하지만, 하나씩 질문하는 게 더 정확한 답변을 받을 수 있어요. 첫 질문이 해결되면 다음 질문을 해주세요.

### Q3. 틀린 답을 받았어요. 어떻게 해야 하나요?
**A:** "이 답이 틀린 것 같은데?"라고 말씀해주시면 다시 확인해드려요. AI도 가끔 실수할 수 있습니다.

### Q4. 문제 사진을 올릴 수 있나요?
**A:** 현재 베타 버전에서는 텍스트만 지원합니다. 문제를 직접 입력해주세요. (이미지 업로드 기능은 곧 추가 예정!)

### Q5. 답만 알려주나요, 아니면 풀이 과정도 알려주나요?
**A:** 풀이 과정을 단계별로 설명해드립니다! "단계별로 설명해줘"라고 요청하시면 더 자세히 알려드려요.

### Q6. 과목을 바꾸고 싶어요.
**A:** 과목 선택 드롭다운 메뉴에서 언제든 변경 가능합니다. 대화 내역은 유지됩니다.

### Q7. 이전 대화 내역을 볼 수 있나요?
**A:** 같은 세션 내에서는 스크롤해서 볼 수 있어요. (대시보드 기능은 곧 추가 예정!)

### Q8. 무료인가요?
**A:** 베타 기간 동안은 완전 무료입니다! 베타 테스터는 정식 출시 후에도 평생 무료 이용 가능합니다.

## 🔧 문제 해결

### AI가 응답하지 않아요
1. 인터넷 연결 확인
2. 페이지 새로고침 (F5)
3. 브라우저 캐시 삭제
4. 다른 브라우저 시도

### 로그인이 안 돼요
1. 이메일 주소 확인
2. 비밀번호 재설정 시도
3. 쿠키 활성화 확인

### 답변이 이상해요
1. 질문을 더 구체적으로 다시 해보세요
2. "다시 설명해줘"라고 요청
3. 과목 선택이 맞는지 확인
