# 검시AI 사용자 온보딩 가이드

> **작성일**: 2026-02-12  
> **목적**: 베타 런칭 시 사용자가 5분 안에 가치를 느끼고 첫 성공 경험을 얻도록 온보딩 설계

---

## 1. First-time User Journey (처음 사용자 여정)

```
┌─────────────┐
│   랜딩      │
│ (홈페이지)  │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 가치 전달   │  ← "24시간 AI 튜터와 검정고시 준비"
│ + CTA       │     메인 메시지, 무료 체험 버튼
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 과목 선택   │  ← 드롭다운에서 관심 과목 선택
│             │    (국어/수학/영어/사회/과학/한국사)
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  첫 질문    │  ← 미리 준비된 질문 예시 3개 제시
│  (AI 튜터)  │     - "개념 설명이 필요하신가요?"
│             │     - "문제 풀이를 도와드릴까요?"
│             │     - "오답 분석을 원하시나요?"
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  AI 응답    │  ← 즉시 답변 (< 3초)
│             │    명확하고 친절한 설명
│             │    관련 문제/예시 제시
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 다음 액션   │  ← 후속 질문 유도:
│ 제안        │    "더 궁금한 점이 있나요?"
│             │    "비슷한 문제를 풀어볼까요?"
│             │    [회원가입하면 학습 기록 저장]
└─────────────┘
```

### 각 단계별 UX 고려사항

#### 1️⃣ 랜딩 (0-10초)
**목표**: 사용자가 "이게 나한테 필요한 서비스구나" 느끼게 하기

- ✅ **명확한 가치 제안**: "검정고시 합격, AI와 함께라면 쉽습니다"
- ✅ **타겟 명시**: 검정고시 준비생이라는 것을 즉시 알 수 있어야 함
- ✅ **마찰 최소화**: 회원가입 없이 바로 "체험해보기" 버튼
- ⚠️ **개선 제안**: 
  - 실제 사용자 후기 1-2개 추가 (신뢰도 ↑)
  - "3분 만에 개념 이해하기" 같은 구체적 시간 명시

#### 2️⃣ 과목 선택 (10-20초)
**목표**: 사용자가 자신의 니즈에 맞는 콘텐츠로 진입

- ✅ **선택권 제공**: 6개 과목 중 선택
- ⚠️ **개선 제안**:
  - "가장 어려워하는 과목부터 시작해보세요" 같은 안내 문구
  - 과목별 인기도/추천 표시 (예: 🔥 수학 - 가장 많이 찾는 과목)
  - 과목 선택 시 해당 과목 미리보기 (예: "수학 - 방정식, 함수, 도형..." )

#### 3️⃣ 첫 질문 (20-40초)
**목표**: 사용자가 무엇을 물어야 할지 막막하지 않게

- ✅ **빠른 선택지 제공**: 개념 설명 / 문제 풀이 / 오답 분석
- ⚠️ **개선 제안**:
  - 실제 질문 예시를 버튼으로 제공 (클릭만 하면 바로 질문)
    - [예: "이차방정식 근의 공식 설명해줘"]
    - [예: "be동사와 일반동사 차이가 뭐야?"]
  - 과목별로 다른 예시 질문 제시 (맞춤형)
  - 텍스트 입력창에 placeholder: "예: 피타고라스 정리가 뭐야?"

#### 4️⃣ AI 응답 (40-90초)
**목표**: "와, 이거 진짜 도움되는데?" 느끼게 하기

- ✅ **빠른 응답 속도**: 3초 이내
- ✅ **친절한 말투**: 반말/존댓말 일관성
- ⚠️ **개선 제안**:
  - **시각적 요소 추가**: 수식은 LaTeX 렌더링, 다이어그램 등
  - **단계별 설명**: 복잡한 개념은 "1단계", "2단계"로 나눠서
  - **이해도 확인**: "이해가 되셨나요?" + [예/아니오] 버튼
  - **관련 콘텐츠 추천**: "이 개념과 관련된 문제를 풀어볼까요?"

#### 5️⃣ 다음 액션 (90-180초)
**목표**: 사용자를 계속 플랫폼에 머물게 하기

- ⚠️ **개선 제안**:
  - **학습 연속성**: "이제 연습문제를 풀어볼까요?"
  - **회원가입 유도**: "회원가입하면 학습 기록이 저장돼요" (너무 강하지 않게)
  - **소셜 프루프**: "오늘 이미 357명이 이 개념을 공부했어요!"
  - **다음 학습 제안**: "다음으로 [X] 개념을 배우면 좋아요"

## 2. 온보딩 체크리스트: 5분 안에 가치 전달하기

### ✅ 첫 30초 체크리스트
- [ ] 사용자가 서비스 목적을 이해했는가? (검정고시 AI 튜터)
- [ ] CTA 버튼이 명확하고 클릭하기 쉬운가?
- [ ] 회원가입 없이 바로 체험 가능한가?

### ✅ 첫 1-2분 체크리스트
- [ ] 과목 선택이 직관적인가?
- [ ] 무엇을 질문해야 할지 막막하지 않은가? (예시 제공)
- [ ] 로딩 시간이 3초 이내인가?

### ✅ 첫 3-5분 체크리스트
- [ ] AI 응답이 실제로 도움이 되는가?
- [ ] 사용자가 "다음에 뭘 해야 할지" 알고 있는가?
- [ ] 회원가입/추가 학습으로 자연스럽게 유도되는가?

### 🎯 첫 성공 경험 설계 (Quick Win)

**목표**: 사용자가 "오, 이거 진짜 괜찮은데?" 느끼는 순간 만들기

#### 시나리오 1: 개념 학습자
1. 사용자: "이차방정식 근의 공식이 뭐야?"
2. AI: 명확한 설명 + 예시 + 시각적 자료
3. AI: "직접 풀어볼까요? 간단한 문제를 드릴게요"
4. 사용자: 문제 풀이 시도
5. **✨ 성공 순간**: "정답이에요! 잘하셨어요 👏" + 해설

#### 시나리오 2: 문제풀이 중심
1. 사용자: "이 문제 풀이 좀 도와줘" (문제 업로드/입력)
2. AI: 단계별 풀이 과정 제시
3. AI: "어느 부분이 어려웠나요?"
4. 사용자: 특정 단계 질문
5. **✨ 성공 순간**: "이제 이해됐어요!" + 비슷한 문제 추천

#### 시나리오 3: 오답 분석
1. 사용자: "이 문제 틀렸는데 왜 틀렸는지 모르겠어"
2. AI: 틀린 이유 분석 + 올바른 접근법
3. AI: "같은 유형 문제를 다시 풀어볼까요?"
4. 사용자: 새 문제 도전
5. **✨ 성공 순간**: 정답! "이제 이 유형은 자신 있어요"

## 3. FAQ - 예상 질문 10개

### Q1. 검시AI는 무료인가요?
**A**: 네, 베타 기간 동안 기본 기능은 무료로 사용할 수 있습니다. 회원가입 후 제한 없이 AI 튜터에게 질문하고, 문제를 풀 수 있어요. 향후 프리미엄 기능(예: 무제한 문제 생성, 맞춤 학습 플랜 등)이 추가될 예정입니다.

### Q2. 회원가입 없이도 사용할 수 있나요?
**A**: 네! 홈페이지에서 "체험해보기"를 클릭하면 바로 AI 튜터를 사용할 수 있습니다. 다만 회원가입을 하시면 학습 기록이 저장되고, 나만의 오답노트와 학습 분석을 받을 수 있어요.

### Q3. 어떤 과목을 지원하나요?
**A**: 검정고시 6개 핵심 과목을 모두 지원합니다:
- 국어
- 수학
- 영어
- 사회
- 과학
- 한국사

### Q4. AI가 틀린 답을 줄 수도 있나요?
**A**: AI는 검정고시 기출문제와 교과서를 기반으로 학습했지만, 100% 완벽하지는 않을 수 있습니다. 답변이 이상하다고 느껴지면 "이 설명이 맞나요?"라고 물어보시거나, 피드백 버튼을 눌러주세요. 여러분의 피드백으로 AI가 더 똑똑해집니다!

### Q5. 문제 사진을 찍어서 올릴 수 있나요?
**A**: 현재는 텍스트 질문만 지원하지만, 곧 사진/이미지 업로드 기능이 추가될 예정입니다! (로드맵에 포함됨)

### Q6. 실제 기출문제를 풀 수 있나요?
**A**: 네! 대시보드에서 "문제풀이" 메뉴로 가면 과목별 기출문제와 유사 문제를 풀 수 있습니다. 틀린 문제는 자동으로 오답노트에 저장됩니다.

### Q7. 다른 사람들은 어떤 질문을 많이 하나요?
**A**: 가장 많이 받는 질문 TOP 3:
1. "이 개념 설명해줘" (개념 이해)
2. "이 문제 풀이 과정 알려줘" (문제풀이)
3. "왜 이게 틀렸는지 모르겠어" (오답 분석)

편하게 무엇이든 물어보세요!

### Q8. 모바일에서도 사용할 수 있나요?
**A**: 네! 검시AI는 모바일, 태블릿, PC 모두 지원합니다. 브라우저만 있으면 언제 어디서나 사용할 수 있어요. (반응형 웹 디자인)

### Q9. 네이버 카페와 어떻게 연결되나요?
**A**: 네이버 카페(준비 중)에서는 다른 검정고시 준비생들과 정보를 공유하고, 검시AI 최신 소식과 학습 팁을 받을 수 있어요. 카페 회원 전용 이벤트와 혜택도 제공될 예정입니다!

### Q10. 학습 진도나 성적을 확인할 수 있나요?
**A**: 네! 회원가입 후 "대시보드"에서 확인할 수 있습니다:
- 📚 **학습 시간**: 과목별 공부한 시간
- 📊 **정답률**: 과목별 문제 정답률
- 📝 **오답노트**: 틀린 문제 복습
- 🎯 **약점 분석**: AI가 분석한 내 약점 영역

## 4. 피드백 수집 계획

### 4.1 수집할 데이터

#### 📊 정량 데이터 (자동 수집)
| 지표 | 측정 방법 | 목적 |
|------|----------|------|
| **완료율** | 랜딩 → 첫 질문 → AI 응답까지 도달한 비율 | 온보딩 펀널 최적화 |
| **이탈률** | 각 단계별 이탈 비율 | 병목 지점 파악 |
| **첫 질문까지 소요 시간** | 랜딩 ~ 첫 질문 제출까지 시간 | 마찰 지점 파악 |
| **세션 시간** | 평균 사용 시간 | 참여도 측정 |
| **질문 개수** | 세션당 평균 질문 수 | 사용 깊이 측정 |
| **재방문율** | 7일 이내 재방문 비율 | 제품 만족도 |
| **회원가입 전환율** | 비로그인 → 로그인 전환 | 가치 인식도 |

#### 💬 정성 데이터 (능동 수집)
| 방법 | 시점 | 질문 |
|------|------|------|
| **첫 사용 후 팝업** | AI 응답 3번째 이후 | "도움이 되셨나요?" ⭐️⭐️⭐️⭐️⭐️ |
| **세션 종료 시 설문** | 이탈 직전 (선택적) | "어떤 점이 아쉬웠나요?" (객관식 3-4개) |
| **회원가입 시 설문** | 회원가입 완료 후 | "검시AI를 어떻게 알게 되셨나요?" (유입 경로) |
| **채팅 피드백 버튼** | 모든 AI 응답 하단 | 👍 도움됨 / 👎 별로 + 이유 (선택) |

### 4.4 피드백 대응 매트릭스

| 피드백 유형 | 대응 방안 | 담당 | 우선순위 |
|------------|----------|------|---------|
| **기술적 오류** (AI 답변 안 옴) | 즉시 수정 | 개발팀 | 🔴 긴급 |
| **AI 답변 품질** (틀린 답변) | 프롬프트 개선 + 검증 | PM + AI팀 | 🟠 높음 |
| **UI/UX 불편** (버튼 안 보임) | 디자인 개선 | 디자인팀 | 🟡 중간 |
| **기능 요청** (사진 업로드) | 로드맵 반영 | PM | 🟢 낮음 |
| **긍정 피드백** (정말 좋아요!) | 케이스 스터디 수집 | 마케팅 | 📘 참고 |

### 4.5 피드백 활용 예시

#### 시나리오: "AI 응답이 너무 어려워요" 피드백 증가
1. **발견**: 지난주 대비 👎 "설명이 어려워요" 30% 증가
2. **분석**: 주로 중학교 기초 과정 질문에서 발생
3. **가설**: AI가 고등학교 수준으로 설명하고 있음
4. **실험**: 프롬프트에 "중학교 수준으로 쉽게 설명" 추가
5. **측정**: A/B 테스트로 만족도 비교
6. **적용**: 효과 있으면 전체 반영

#### 시나리오: 첫 질문까지 시간이 너무 김 (평균 50초)
1. **발견**: 벤치마크 대비 2배 느림 (목표: 20초)
2. **분석**: 사용자가 뭘 질문해야 할지 모름
3. **가설**: 예시 질문 버튼이 없어서 막막함
4. **실험**: 클릭 가능한 예시 질문 3개 추가
5. **측정**: 첫 질문까지 시간 변화 추적
6. **결과**: 평균 25초로 감소 → 정식 반영

## 5. 온보딩 플로우 다이어그램 (텍스트 버전)

```
┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃          검시AI 온보딩 플로우 (베타)           ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

🌐 STEP 1: 랜딩 (0-10초)
├─ 사용자 도착 (네이버 카페, SNS, 검색 등)
├─ 헤드라인: "검정고시 합격, AI와 함께라면 쉽습니다"
├─ 서브카피: "24시간 언제든지, 나만의 AI 튜터와 함께"
├─ CTA 버튼: [무료로 시작하기] [체험해보기]
└─ 스크롤: 기능 소개 (AI 맞춤 학습, 24시간 질문 가능, 학습 분석)

      │
      ▼

🎯 STEP 2: 진입점 선택 (10-15초)
├─ A 경로: "무료로 시작하기" → 회원가입
├─ B 경로: "체험해보기" → 바로 AI 튜터 (추천!)
└─ 🔍 목표: 마찰 최소화 (B 경로 80% 유도)

      │ (B 경로 - 체험해보기 클릭)
      ▼

📚 STEP 3: 과목 선택 (15-25초)
├─ 드롭다운: 전체 / 국어 / 수학 / 영어 / 사회 / 과학 / 한국사
├─ 추천 과목 표시: 🔥 수학 (가장 인기) / ⚡ 영어 (빠른 개념 학습)
├─ 툴팁: "가장 어려워하는 과목부터 시작해보세요"
└─ 선택 완료 → AI 튜터 화면 진입

      │
      ▼

💬 STEP 4: 첫 질문 (25-40초)
├─ AI 인사 메시지:
│   "안녕하세요! 저는 검정고시 전문 AI 튜터예요.
│    무엇이든 편하게 물어보세요!"
├─ 빠른 선택 (클릭 가능한 버튼):
│   [개념 설명이 필요해요]
│   [문제 풀이 도와주세요]
│   [오답 분석 부탁해요]
├─ 실제 질문 예시 버튼 (과목별 맞춤):
│   수학: [이차방정식 근의 공식 설명해줘]
│   영어: [be동사와 일반동사 차이가 뭐야?]
│   국어: [주체높임법이 뭐야?]
└─ 또는 직접 입력: "예: 피타고라스 정리가 뭐야?"

      │ (질문 제출)
      ▼

🤖 STEP 5: AI 응답 (40-90초)
├─ 로딩: "생각 중..." (3초 이내)
├─ 답변 생성:
│   ✅ 친절하고 명확한 설명
│   ✅ 단계별 정리 (1단계, 2단계...)
│   ✅ 예시 포함
│   ✅ 시각적 요소 (수식, 다이어그램)
├─ 이해도 확인: "이해가 되셨나요?" [네] [아니요, 다시 설명해줘]
└─ 피드백 버튼: 👍 도움됨 / 👎 아쉬워요

      │
      ▼

🎓 STEP 6: 다음 액션 제안 (90-180초)
├─ A 경로: 추가 질문
│   "더 궁금한 점이 있나요?"
│   "이 개념과 관련해서 더 알고 싶은 게 있나요?"
├─ B 경로: 연습 문제
│   "비슷한 문제를 풀어볼까요?"
│   [예] → 문제 생성 → 풀이 → 해설
├─ C 경로: 회원가입 유도 (soft pitch)
│   💾 "회원가입하면 학습 기록이 저장돼요"
│   📊 "지금까지 3개 질문을 하셨어요. 회원가입하고 계속 배워보세요!"
└─ D 경로: 소셜 프루프
    "오늘 이미 357명이 이 개념을 공부했어요!"

      │
      ▼

✨ STEP 7: Quick Win (첫 성공 경험) (3-5분)
├─ 시나리오 A: 개념 이해 → 간단한 퀴즈 풀이 → 정답! 🎉
├─ 시나리오 B: 문제 풀이 도움 → 단계별 해설 → "이제 이해됐어요!"
├─ 시나리오 C: 오답 분석 → 왜 틀렸는지 파악 → 비슷한 문제 다시 도전 → 성공!
└─ 보상: "잘하셨어요! 👏" + 다음 학습 추천

      │
      ▼

🔁 STEP 8: 참여 루프 (5분+)
├─ 계속 질문하기 (비로그인도 가능)
├─ 또는 회원가입:
│   → 학습 기록 저장
│   → 대시보드에서 진도 확인
│   → 맞춤형 학습 플랜
└─ 이탈 시: 이탈 설문 → 개선 피드백 수집

┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  목표 지표                                      ┃
┃  ✅ 첫 질문까지 도달율: 70%+                     ┃
┃  ✅ 세션당 평균 질문 수: 3개+                    ┃
┃  ✅ 만족도 (⭐️4점 이상): 80%+                  ┃
┃  ✅ 회원가입 전환율: 20%+                        ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
```
