## 3. 첫 AI 튜터 인터랙션 스크립트

### Scene 1: AI 튜터 등장 (애니메이션)

```
[AI 아바타가 부드럽게 나타남]

AI: 안녕하세요! 저는 당신의 AI 과외 선생님이에요 👋

반갑습니다! 수학과 영어 공부를 도와드리게 되어 기쁩니다.

먼저, 지금 가장 도움이 필요한 부분이 무엇인지 
편하게 이야기해 주세요.

예를 들면:
• "미적분 개념이 헷갈려요"
• "영어 독해 속도를 높이고 싶어요"
• "최근에 틀린 문제를 다시 풀고 싶어요"
```

**유저 입력창 placeholder:**
```
무엇을 도와드릴까요? 편하게 이야기해주세요...
```

---

### Scene 2: 첫 문제 제시 (예시: "미적분 개념이 헷갈려요" 입력 시)

```
AI: 미적분이 어렵게 느껴지시는군요. 괜찮아요! 
함께 차근차근 정복해봐요 💪

먼저 현재 이해도를 확인하고, 
딱 맞는 난이도부터 시작할게요.

간단한 문제 하나 풀어볼까요?

───────────────────────────
📝 워밍업 문제

함수 f(x) = x² + 2x의 x=1에서의 미분계수를 구하시오.

[풀이 시작하기]
[힌트가 필요해요]
[개념부터 설명해주세요]
───────────────────────────
```

---

### Scene 3: 사용자 답변 후 피드백

**정답인 경우:**
```
AI: 완벽합니다! 🎉

f'(x) = 2x + 2이고, f'(1) = 4가 맞아요.

미분의 기본 공식을 정확히 이해하고 계시네요!
이제 조금 더 응용된 문제로 넘어가볼까요?

─────────────────
💡 학습 팁
미분계수는 '순간 변화율'을 의미해요. 
이 개념을 꼭 기억해두세요!
─────────────────

[다음 문제로] [개념 더 배우기]
```

**오답인 경우:**
```
AI: 좋은 시도였어요! 하지만 조금 다르게 접근해봐야 할 것 같아요.

함께 단계별로 풀어볼까요?

1️⃣ 먼저 f(x) = x² + 2x를 미분하면?
   → f'(x) = 2x + 2 (거듭제곱 미분 법칙)

2️⃣ x=1을 대입하면?
   → f'(1) = 2(1) + 2 = 4

이해가 되셨나요? 비슷한 문제를 하나 더 풀어보면
확실히 잡힐 거예요!

[비슷한 문제 풀기] [개념 다시 설명 듣기]
```

```
✓ 미분계수의 의미

─────────────────
📊 오늘의 학습
• 문제 해결: 3/4개 정답

─────────────────

💬 베타 피드백 부탁드려요!
오늘 수업이 어떠셨나요? 
여러분의 의견이 검시AI를 더 좋게 만듭니다.

[피드백 남기기] [내일 이어서 공부하기]
```
