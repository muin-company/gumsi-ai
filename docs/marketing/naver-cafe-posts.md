혹시 저처럼 독학하시는 분들 계시면 한번 써보세요!
괜히 고민만 하지 말고 일단 써보는 게 나은 것 같아요.

🔗 https://gumsi-ai.vercel.app

(참고: 전 아무 관련 없는 사람이고, 그냥 써보고 좋아서 공유하는 거예요)

## 🎯 버전 5: 독학/자격증 카페용 (팁 공유 톤)

### 제목
[꿀팁] 혼자 공부할 때 AI 활용하는 법

### 본문
```
독학러 3년차인데, 최근에 AI 활용법 터득해서 공유드립니다!

## 독학의 최대 적: "물어볼 사람 없음"

학원/과외 없이 혼자 공부하면 제일 답답한 게 이거죠.
문제집 풀다가 막히면 그냥 넘어가거나, 구글링하면서 시간 낭비...

요즘은 AI 튜터 써서 이 문제 해결하고 있어요.

## 제가 쓰는 방법

**1. 일단 혼자 풀어본다**
- AI한테 물어보기 전에 일단 내 힘으로 최대한

**2. 막히면 AI한테 질문**
- "이 문제 풀이 과정 설명해줘"
- "이 개념 중학생도 이해할 수 있게 설명해줘"
- 사진 찍어서 올리면 더 빠름

**3. AI 설명 들으면서 이해**
- 이해 안 되면 "더 쉽게 설명해줘" 다시 물어봄
- "예시 들어줘" 하면 예시로 설명해줌

**4. 비슷한 문제 다시 풀어보기**
- 이해했다고 끝이 아님! 꼭 직접 풀어봐야 함

## 제가 쓰는 AI

검시AI (https://gumsi-ai.vercel.app) 라고, 검정고시용인데
중고등 과정 전반적으로 다 돼서 그냥 쓰고 있어요.

- 완전 무료
- 회원가입 필요 없음
- 24시간 사용 가능

## 주의사항 ⚠️

AI는 보조 도구일 뿐! 100% 믿으면 안 됩니다.
- 가끔 틀린 답 줌 (특히 복잡한 문제)
- 중요한 건 교과서/참고서로 확인 필수
- AI가 공부를 대신해주는 건 아님 (결국 내가 이해해야 함)

## 결론

독학 = 혼자 하는 게 아니라 "내 페이스대로 하는 것"
AI 튜터 활용하면 비용 부담 없이 질문할 수 있어서
독학이 훨씬 수월해집니다!

다들 공부 화이팅! 💪
```

## ⚠️ 주의사항

### 1. 광고 티 제거
- "홍보", "광고", "추천" 같은 단어 피하기
- "제가 써봤는데", "공유드려요" 같은 자연스러운 톤
- 과도한 칭찬 피하기 (장단점 모두 언급)

### 2. 카페 규정 준수
- 게시 전 반드시 카페 규정 확인
- 광고 금지 카페는 절대 피하기
- 관리자 눈치 보면서 조심스럽게

### 3. 댓글 관리
- 게시 후 댓글 모니터링 필수
- 질문에는 친절하게 답변
- 부정적 반응에는 방어적으로 반응하지 말기

### 4. 스팸 방지
- 같은 카페에 여러 번 게시 금지
- 하루에 너무 많은 카페에 게시하지 않기 (3-5개 정도)
- 시간 간격 두고 게시 (한꺼번에 도배하면 티남)

### 5. 법적 이슈
- "완전 무료" 사실 확인 (나중에 유료 전환 시 문제될 수 있음)
- 과장된 효과 주장 금지 ("성적 100% 올라감" 같은 거)
- AI 한계 명시 (가끔 틀린 답 줄 수 있음)
