- 부제: `검시AI 커뮤니티 - 수능 공부의 새로운 파트너`

**서버 아이콘**
- 크기: 512x512px (권장)
- 포맷: PNG, GIF (애니메이션 가능)
- 디자인: GumsiAI 로고 (간결한 버전)
- 배경: 투명 또는 브랜드 컬러
- 파일 위치: `~/muin-company/gumsi-ai/assets/discord-icon.png`

**서버 배너**
- 크기: 960x540px (권장)
- 포맷: PNG, JPG
- 디자인 요소:
  - GumsiAI 로고 + 슬로건
  - "수능 공부의 AI 파트너"
  - 깔끔하고 전문적인 느낌
  - 브랜드 컬러 활용
- 파일 위치: `~/muin-company/gumsi-ai/assets/discord-banner.png`

**서버 설명**
```
GumsiAI 공식 커뮤니티입니다! 🎓

AI와 함께하는 스마트한 수능 공부

- 과목별 질문 & 토론
- 함께 성장하는 학습 커뮤니티

https://gumsi.ai
```

### 1.2 서버 설정

**Vanity URL**
- 목표: `discord.gg/gumsi` 또는 `discord.gg/gumsi-ai`
- 조건: 서버 부스트 레벨 3 필요 (초기에는 불가능할 수 있음)
- 임시 초대 링크: 생성 후 문서화

**Verification Level (인증 레벨)**
- 설정: **Medium (중간)**
  - Discord 계정이 5분 이상 된 사용자만 참여
  - 이메일 인증 필수
- 이유: 스팸봇 차단 & 품질 유지, 너무 높으면 신규 유저 진입장벽

**Explicit Content Filter (콘텐츠 필터)**
- 설정: **Scan messages from members without a role (역할 없는 멤버의 메시지 스캔)**
- 이유: 신규 가입자 보호, 기존 멤버는 자율성 부여

**2FA Requirement (2단계 인증)**
- 모더레이터 이상: **필수**
- 일반 멤버: 선택

**Default Notifications (기본 알림)**
- 설정: **Only @mentions (멘션만)**
- 이유: 활발한 커뮤니티에서 알림 피로 방지

**System Messages Channel**
- 채널: `#welcome`
- 활성화 메시지:
  - ✅ 새 멤버 가입 환영
  - ✅ 서버 부스트 알림
  - ❌ 기타 시스템 메시지 (잡음 방지)

### 1.3 커뮤니티 기능 활성화

**Community Server 설정**
- Discovery 활성화 (서버 디렉토리 등재)
- Rules Screening 활성화 (가입 시 규칙 동의)
- Announcement Channels 설정

**Server Insights**
- 활성화하여 성장 지표 추적
- 주간 리포트 검토

### 2.2 채널별 상세 설정

#### 📢 공지 카테고리

**#📣announcements (공지사항)**
- 권한: 읽기 전용 (Admin/Mod만 작성)
- 용도: 중요 공지, 정책 변경, 서비스 업데이트
- 설정: Announcement Channel (다른 서버에서 팔로우 가능)
- 토픽: `GumsiAI 공식 공지사항 | 중요한 소식을 전합니다`
- 알림: @everyone 사용 가능 (신중하게)

**#🔔updates (업데이트)**
- 권한: 읽기 전용
- 용도: 제품 업데이트, 새 기능, 패치 노트
- 토픽: `GumsiAI 업데이트 & 패치 노트 | 새로운 기능을 확인하세요`
- 빈도: 주 1-2회

**#🎉events (이벤트)**
- 권한: 읽기 전용
- 용도: 이벤트 안내, 스터디 세션, Q&A 일정
- 토픽: `커뮤니티 이벤트 & 스터디 세션 안내`
- Discord Events 기능 활용

#### 🚪 시작 카테고리

**#👋welcome (환영)**
- 권한: 읽기 전용 (봇 자동 메시지)
- 용도: 신규 멤버 환영, 가이드 링크
- 설정: System Messages Channel로 지정
- 토픽: `GumsiAI 커뮤니티에 오신 것을 환영합니다! 🎓`

**#📜rules (규칙)**
- 권한: 읽기 전용
- 용도: 서버 규칙, 행동 강령
- 설정: Rules Channel (Community 기능)
- 토픽: `서버 규칙 | 모두가 즐거운 커뮤니티를 위해`
- 내용: 아래 4.2 참고

**#🤝introductions (자기소개)**
- 권한: @Member 이상
- 용도: 신규 멤버 자기소개
- 토픽: `자기소개 채널 | 간단히 인사해주세요! 😊`
- 슬로우모드: 30초
- 템플릿 제공:
  ```
  안녕하세요! 👋
  - 닉네임:
  - 수험생/재수생/N수생:
  - 목표 대학/학과:
  - 관심 과목:
  - 한마디:
  ```

## 7. 모더레이션

### 7.1 모더레이션 원칙

1. **공정성:** 모든 멤버를 동등하게 대우
2. **투명성:** 조치 이유 명확히 설명
3. **일관성:** 같은 위반에는 같은 조치
4. **교육적:** 처벌보다 교육 우선
5. **신속성:** 위반 사항 즉시 처리

### 7.2 위반 유형 및 조치

#### Level 1: 경미한 위반

**유형:**
- 잘못된 채널 사용
- 가벼운 스팸 (의도 아님)
- 과도한 대문자 사용
- 주제 이탈

**조치:**
- 🔔 부드러운 경고 (메시지 삭제 + 댓글)
- 올바른 사용 안내
- 기록하지 않음

**예시 응답:**
```
안녕하세요! 이 채널은 [용도]를 위한 채널이에요. 
[해당 내용]은 <#채널명>에 올려주시면 감사하겠습니다! 😊
```

#### Level 2: 중간 위반

**유형:**
- 반복적인 채널 오용
- 가벼운 욕설 (의도적)
- 무단 광고
- 다른 서버 초대 링크

**조치:**
- ⚠️ 공식 경고 (Dyno `?warn`)
- 메시지 삭제
- DM으로 경고 사유 설명
- 기록 유지

**예시 응답:**
```
[DM]
안녕하세요, GumsiAI Mod입니다.

귀하의 메시지는 서버 규칙 [X조]를 위반하여 삭제되었습니다:
"[메시지 내용]"

이는 첫 번째 경고입니다. 추가 위반 시 타임아웃 또는 강퇴 조치가 있을 수 있습니다.

규칙: <#rules>
문의 사항이 있으시면 답장해주세요.
```

#### Level 3: 심각한 위반

**유형:**
- 심한 욕설, 비방
- 괴롭힘, 반복적 위반
- 스팸봇 의심
- 규칙 무시 (3회 경고 후)

**조치:**
- 🔇 타임아웃 (1시간 - 7일)
- 기간은 위반 심각도에 따라
- DM으로 사유 설명
- 이의 제기 가능 안내

**예시 응답:**
```
[DM]
안녕하세요, GumsiAI Mod입니다.

귀하는 서버 규칙 [X조] 심각한 위반으로 [기간] 타임아웃되었습니다:
"[사유]"

타임아웃 기간: [시간]
해제 시각: [UTC 시간]

이의 제기를 원하시면 Admin에게 DM 주시기 바랍니다.
```

#### Level 4: 매우 심각한 위반

**유형:**
- 불법 콘텐츠 (NSFW, 저작권)
- 심각한 괴롭힘, 협박
- 개인정보 유출
- 악의적 링크 (피싱, 멀웨어)

**조치:**
- 🚪 강퇴 (Kick) 또는 🔨 차단 (Ban)
- 즉시 조치 (사전 경고 없음)
- 증거 보존 (#mod-logs)
- 필요 시 Discord Trust & Safety에 신고

**차단 vs 강퇴:**
- 강퇴: 재가입 가능 (신중히 재고 기회)
- 차단: 재가입 불가 (악의적 행위)

### 7.3 모더레이터 가이드라인

#### 일상 업무

**매일:**
- 채널 순회 (1회 이상)
- #mod-logs 확인
- 신고 처리
- 스팸/광고 제거

**주간:**
- 경고 내역 검토
- 모더레이터 회의 (30분)
- FAQ 업데이트
- 커뮤니티 피드백 수집

**월간:**
- 규칙 효과성 검토
- Mod 권한 감사
- 통계 리포트 작성
- Admin과 정책 논의

#### 모더레이터 명령어

**Dyno:**
- `?warn @user [사유]` - 경고
- `?warnings @user` - 경고 내역
- `?timeout @user [시간]` - 타임아웃
- `?kick @user [사유]` - 강퇴
- `?ban @user [사유]` - 차단
- `?unban @user` - 차단 해제
- `?purge [개수]` - 메시지 삭제
- `?lock #channel` - 채널 잠금

**MEE6:**
- `!clearwarn @user` - 경고 삭제
- `!infractions @user` - 위반 내역

#### 어려운 상황 대처

**분쟁 중재:**
1. 양측 이야기 듣기
2. 규칙 기반 판단
3. 중립 유지
4. 필요 시 Admin 에스컬레이션

**애매한 경우:**
- 다른 Mod와 상의
- Admin에게 문의
- 커뮤니티 의견 수렴 (투표)

**Mod 실수 시:**
- 솔직하게 인정
- 사과 및 조치 철회
- 재발 방지 논의

#### 자기 관리

- 번아웃 주의
- 휴식 필요 시 알리기
- 감정적 대응 피하기
- 의문 있으면 상의하기

### 7.4 신고 처리 프로세스

1. **신고 접수:** DM 또는 #mod-chat
2. **증거 수집:** 스크린샷, 로그
3. **사실 확인:** 맥락 파악
4. **조치 결정:** 규칙 기반
5. **조치 실행:** 명령어 사용
6. **기록:** #mod-logs에 자동 기록 + 노트
7. **피드백:** 신고자에게 결과 알림

## 10. Metrics & KPIs

### 10.1 주요 지표

#### 성장 지표

**일일 가입자 수 (Daily Sign-ups)**
- 목표: Week 1: 10-20명/일, Week 2: 20-50명/일
- 측정: Server Insights, Statbot
- 분석: 홍보 효과, 유입 경로

**총 멤버 수 (Total Members)**
- 목표: Week 1: 100명, Month 1: 500명, Month 3: 2000명
- 측정: Server Insights
- 마일스톤: 100, 500, 1000, 2000, 5000명

**유입 경로 (Traffic Sources)**
- 측정: 초대 링크별 추적
- 분석: 어떤 채널이 효과적인지
- 최적화: 효과적인 채널에 집중

#### 활동 지표

**활성 사용자 (DAU/MAU)**
- DAU: 일일 활성 사용자 (메시지 1개 이상)
- MAU: 월간 활성 사용자
- 목표: DAU/MAU > 20% (건강한 커뮤니티)
- 측정: Statbot, MEE6

**메시지 수 (Messages per Day)**
- 목표: Week 1: 50+, Week 2: 100+, Month 1: 300+
- 측정: Statbot
- 분석: 채널별 활동량

**음성 채널 사용률 (Voice Usage)**
- 측정: 일일 음성 채널 사용 시간
- 목표: 주간 스터디 세션 10+ 참여자
- 분석: 어떤 시간대가 인기있는지

#### 참여 지표

**평균 세션 시간 (Avg Session Duration)**
- 측정: 멤버가 온라인 상태로 머무는 평균 시간
- 목표: 30분 이상
- 도구: 분석 봇 (커스텀)

**리액션 & 스레드 (Engagement)**
- 측정: 메시지당 평균 리액션 수, 스레드 생성 수
- 목표: 리액션 비율 > 10%, 스레드 주 5+ 개
- 분석: 콘텐츠 품질 지표

**이벤트 참여율 (Event Attendance)**
- 측정: 이벤트 참여자 / 총 멤버
- 목표: 10% 이상
- 분석: 어떤 이벤트가 인기있는지

#### 유지 지표

**Retention Rate (재방문율)**
- 측정: Day 1, Day 7, Day 30 재방문율
- 목표: Day 7 > 40%, Day 30 > 20%
- 분석: 초기 경험이 중요

**이탈률 (Churn Rate)**
- 측정: 탈퇴자 / 총 멤버
- 목표: < 10% (월간)
- 분석: 이탈 이유 파악 (가능 시)

#### 품질 지표

**모더레이션 액션 (Mod Actions)**
- 측정: 경고, 타임아웃, 강퇴 수
- 목표: < 5% 멤버가 조치 대상
- 분석: 커뮤니티 건강도

**신고 건수 (Reports)**
- 측정: 일일 신고 건수
- 목표: 감소 추세
- 분석: 자정 능력

**사용자 만족도 (User Satisfaction)**
- 측정: 월간 설문 (1-5점)
- 목표: 평균 4점 이상
- 도구: Google Form + #feedback

### 10.2 데이터 수집 도구

**Discord 내장:**
