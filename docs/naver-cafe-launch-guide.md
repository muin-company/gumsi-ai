#### 1. 검정고시 합격하기

**카페 정보:**
- URL: `cafe.naver.com/ged123` (예시 - 실제 URL 확인 필요)
- 멤버 수: ~10,000명 (추정)
- 활동도: 높음 (일 50+ 게시물)
- 규정: 홍보 가능 여부 확인 필요

**타겟 게시판:**
- 자유게시판
- 학습 정보 공유
- 질문/답변

**포스팅 전략:**
- 제목: `[무료] 검정고시 AI 튜터 베타 테스터 모집 (선착순 100명)`
- 톤앤매너: 진솔하고 친근하게 (동료 수험생처럼)
- 강조점: 24시간 질문 가능, 학원비 부담 없음, 독학 효율 증대

**예상 반응:**
- 긍정적: "오, 이거 좋을 것 같은데?" / "신청했어요!"
- 중립적: "진짜 무료야?" / "AI 답변 정확해?"
- 부정적: "또 홍보글이네..." / "AI로 공부가 돼?"

**대응 전략:**
- 긍정적: 감사 댓글 + Discord 초대
- 중립적: FAQ 링크 + 상세 설명
- 부정적: 겸손한 태도 + 베타 테스트 취지 설명

## 카페별 포스팅 전략

### 1. 카페 규칙 확인 프로세스

**사전 조사 체크리스트:**

```
각 카페별로 다음 항목 확인:

[ ] 홍보 게시글 허용 여부
  - 허용: 자유롭게 포스팅
  - 제한적 허용: 홍보 게시판만 가능
  - 금지: 포스팅 금지 (Skip)

[ ] 홍보 게시글 형식 규정
  - 제목에 [홍보] 태그 필수 여부
  - 본문 길이 제한
  - 이미지 개수 제한
  - 링크 개수 제한

[ ] 포스팅 빈도 제한
  - 하루 1회
  - 일주일 1회
  - 제한 없음

[ ] 관리자 사전 승인 필요 여부
  - 필요: 관리자 쪽지로 승인 요청
  - 불필요: 바로 포스팅

[ ] 금지어/민감 표현
  - "무료" 사용 가능 여부
  - "돈 벌기" 등 부정적 연상 단어
  - 과도한 이모지 사용

[ ] 기타 특이사항
  - 카페별 문화/분위기
  - 과거 유사 홍보글 반응
```

**조사 방법:**

1. **카페 공지사항 정독**
   - "카페 이용 규칙"
   - "홍보 관련 공지"
   - "게시판 이용 안내"

2. **최근 홍보 게시글 분석**
   - 검색어: "베타", "무료", "모집"
   - 긍정적 반응 vs 부정적 반응
