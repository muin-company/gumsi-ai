# 검시AI 베타 테스터 FAQ

베타 테스터 여러분께 자주 묻는 질문과 답변

---

## 🎯 베타 프로그램 관련

### Q: 베타 기간은 언제까지인가요?
**A:** 2026년 2-3월 약 4-6주 예정입니다. 100명 선착순 모집 후 대기 명단이 운영됩니다.

### Q: 베타 테스터 혜택이 뭔가요?
**A:**
1. **평생 무료 이용권** - 정식 출시 후에도 모든 기능 무료
2. **우선 기능 추가** - 요청한 기능 먼저 개발
3. **학습 자료 제공** - 검정고시 핵심 요약본 무료
4. **베타 커뮤니티** - 다른 테스터들과 정보 공유

### Q: 베타 테스터로 선정되면 할 일이 뭔가요?
**A:**
1. **최소 주 3회 이상 사용** (매일 추천)
2. **피드백 제출** (좋았던 점, 개선할 점)
3. **버그 리포트** (발견한 문제 즉시 공유)
4. **(선택) 사용 후기 작성** - 다른 학생들에게 도움

### Q: 베타 기간 중 유료로 전환되나요?
**A:** 절대 안 됩니다! 베타 테스터는 **평생 무료**입니다.

## 🔧 기술 관련

### Q: 어떤 기기에서 사용할 수 있나요?
**A:**
- ✅ PC (Windows, Mac, Linux)
- ✅ 스마트폰 (Android, iOS)
- ✅ 태블릿
- ✅ 모든 최신 브라우저 (Chrome, Safari, Firefox, Edge)

### Q: 앱이 있나요?
**A:** 아직 앱은 없지만, 웹사이트를 홈 화면에 추가하면 앱처럼 사용 가능합니다!
- **iPhone**: Safari에서 공유 버튼 → "홈 화면에 추가"
- **Android**: Chrome에서 메뉴 → "홈 화면에 추가"

### Q: 오프라인에서도 사용할 수 있나요?
**A:** 현재는 인터넷 연결이 필요합니다. 오프라인 모드는 향후 고려 중입니다.

### Q: 데이터는 안전한가요?
**A:** 네! 모든 데이터는 암호화되어 안전하게 저장됩니다. 개인정보는 절대 외부에 공유되지 않습니다.

## 💡 기능 관련

### Q: AI가 틀린 답을 주면 어떻게 하나요?
**A:**
1. "이 답이 틀린 것 같은데?"라고 말씀해주세요
2. AI가 다시 확인하고 수정된 답을 드립니다
3. 심각한 오류는 즉시 support@muin.company로 제보

### Q: 이미지(사진)를 올릴 수 있나요?
**A:** 베타 버전에서는 텍스트만 가능합니다. 문제나 그림은 직접 입력해주세요.
- 📸 **이미지 업로드 기능**: 3월 중 추가 예정!

### Q: 대화 내역이 저장되나요?
**A:**
- 같은 세션: 저장됨 (새로고침하면 사라짐)
- 다른 세션: 현재는 저장 안 됨
- 📚 **대시보드 기능**: 3월 중 추가 예정 (모든 대화 저장)

### Q: 음성으로 질문할 수 있나요?
**A:** 베타 버전에서는 텍스트만 가능합니다.
- 🎤 **음성 입력**: 향후 추가 검토 중

### Q: 과목별로 다른 AI 선생님이 있나요?
**A:** 현재는 하나의 AI가 모든 과목을 담당합니다. 과목을 선택하면 AI가 그 과목 전문가 모드로 전환됩니다.

## 📚 학습 관련

### Q: 검정고시 기출문제도 풀 수 있나요?
**A:**
- 문제를 입력하면 풀이 도움 가능
- 기출문제 DB는 곧 추가 예정
- **모의고사 기능**: 3-4월 추가 예정

### Q: 어떤 검정고시를 지원하나요?
**A:**
- ✅ 중학 검정고시 (국어, 수학, 영어, 사회, 과학)
- ✅ 고등 검정고시 (국어, 수학, 영어, 한국사, 사회, 과학)

### Q: 수능도 준비할 수 있나요?
**A:** 현재는 검정고시 중심이지만, 기본 개념은 수능과 동일합니다. 수능 특화 기능은 향후 추가 예정입니다.

### Q: 문제집 추천해주나요?
**A:** AI 튜터에게 "검정고시 수학 문제집 추천해줘"라고 물어보시면 추천 드립니다!

### Q: 학습 계획도 세워주나요?
**A:** 네! "한 달 수학 학습 계획 세워줘"라고 요청하시면 맞춤 계획을 제공합니다.
- 📅 **스마트 스케줄러**: 3-4월 추가 예정 (진도 자동 추적)

## 🐛 문제 해결

### Q: 로그인이 안 돼요.
**A:**
1. 이메일 주소 다시 확인
2. 비밀번호 재설정 시도
3. 브라우저 쿠키 활성화 확인
4. 시크릿/사생활 모드가 아닌지 확인

### Q: AI가 응답을 안 해요.
**A:**
1. 인터넷 연결 확인
2. 페이지 새로고침 (F5 또는 ⌘+R)
3. 다른 브라우저에서 시도
4. 10분 후 다시 시도 (서버 점검 중일 수 있음)

### Q: 화면이 깨져 보여요.
**A:**
1. 브라우저 캐시 삭제
2. 최신 브라우저 버전 확인
3. 화면 확대/축소 100% 확인
4. 다른 기기에서 시도

### Q: 너무 느려요.
**A:**
1. 인터넷 속도 확인
2. 다른 탭들 닫기
3. 브라우저 재시작
4. 모바일 데이터 대신 Wi-Fi 사용

## 💬 피드백 관련

### Q: 피드백은 어떻게 보내나요?
**A:**
- **이메일**: support@muin.company
- **제목**: [베타 피드백] 제목
- **내용**: 상황 설명 + 스크린샷(선택)

### Q: 어떤 피드백이 도움이 되나요?
**A:**
1. **구체적인 상황**: "수학 문제 풀이 요청했을 때..."
2. **기대했던 것**: "단계별 설명을 원했는데..."
3. **실제 결과**: "답만 알려줬어요"
4. **개선 아이디어**: "이렇게 하면 더 좋을 것 같아요"

### Q: 제안한 기능이 언제 추가되나요?
**A:**
- **긴급 버그**: 1-2일 내 수정
- **작은 개선**: 1-2주 내 반영
- **큰 기능**: 한 달 이상 소요 가능
- 진행 상황은 이메일로 공유드립니다!

### Q: 피드백 보상이 있나요?
**A:**
- 베타 테스터 자체가 이미 큰 혜택! (평생 무료)
- 우수 피드백 제공자에게는 추가 혜택 검토 중
